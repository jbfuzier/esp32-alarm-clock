# ESP32 Alarm Clock

This repository contains the code and setup instructions for building an ESP32-based Alarm Clock.

## Features
- Adjustable alarm times
- Snooze function
- Display of current time
- User-friendly interface
- Adjustable volume
- Supports multiple alarms

## Setup Instructions
1. ### Hardware Requirements
   - ESP32 Development Board
   - Breadboard
   - Jumper Wires
   - Optional: LCD Display (I2C supported)

2. ### Software Requirements
   - Arduino IDE (with ESP32 board support)
   - Required Libraries:
     - <Library_1>
     - <Library_2>

3. ### Steps to Set Up
   - Connect the ESP32 board to your computer.
   - Open the Arduino IDE and set up the board configurations.
   - Clone this repository to your local machine.
   - Install the required libraries if not already done.
   - Upload the code to your ESP32.
   - Assemble the circuit as per the wiring diagram provided.

4. ### Usage
   - Power on the device and set the time using the buttons provided.
   - Set alarms using the interface.
   - Use the snooze button as required.

Enjoy your ESP32 Alarm Clock!
